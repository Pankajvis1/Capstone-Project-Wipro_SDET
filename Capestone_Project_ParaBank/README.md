# 🏦 ParaBank Automation Testing Framework

## 🚀 Enterprise-Level Selenium Automation Framework

Built using **Java • Selenium WebDriver • TestNG • Maven • Page Object
Model • Log4j2 • Extent Reports • Allure • Docker • Jenkins**

------------------------------------------------------------------------

## 📖 About The Project

This project is a comprehensive Selenium Automation Framework developed
for the **ParaBank Demo Banking Application**.

### Key Features

-   ✅ Page Object Model (POM)
-   ✅ Driver Factory Pattern
-   ✅ TestNG Framework
-   ✅ Cross Browser Testing
-   ✅ Data Driven Testing
-   ✅ CSV Integration
-   ✅ Log4j2 Logging
-   ✅ Extent Reporting
-   ✅ Allure Reporting
-   ✅ Screenshot Capture
-   ✅ Maven Build Management
-   ✅ Jenkins CI/CD Integration
-   ✅ Docker Support

------------------------------------------------------------------------

## 🧪 Test Coverage

### Positive Scenarios

-   User Registration
-   User Login
-   Open New Account
-   Account Overview
-   Transfer Funds
-   Bill Payment
-   Request Loan
-   Find Transactions
-   Update Contact Information
-   Logout

### Negative Scenarios

-   Invalid Login
-   Empty Credentials
-   Invalid Bill Payment
-   Invalid Loan Request
-   Invalid Transfer Amount
-   Same Account Transfer Validation
-   Invalid Transaction Search
-   Invalid Contact Information
